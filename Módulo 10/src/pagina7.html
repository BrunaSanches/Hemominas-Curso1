Construção de banco de dados
Dados secundários

A construção de banco de dados com dados secundários é uma alternativa para estabelecimento de saúde com agencias transfusionais incipientes ou nos quais ainda não foi possível realizada a coleta de dados primário e a construção do perfil transfusional do estabelecimento.

A construção de um banco de dados com dados secundários pode ser útil também para a comparação do perfil transfusional do estabelecimento de saúde com outro estabelecimento integrante do Sistema Único de Saúde.
